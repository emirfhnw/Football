# Evaluation Tasks

Use these tasks to evaluate whether participants can understand and use the dashboard.

## Task 1

Find the most common goal build-up type.

## Task 2

Find the team with the highest average passes before goal.

## Task 3

Select a Spain goal and explain the build-up sequence using the pitch and timeline.

## Task 4

Use the scatterplot to inspect a long build-up goal.

## Task 5

Compare whether longer build-ups also take more time.

## Task 6

Reset filters and find a quick attack goal.

## Observation Notes

- Did the participant understand quick, medium and long build-ups?
- Did the participant notice the event-data limitation?
- Did the pitch and timeline help explain the selected goal?
- Did linked views behave as expected?
- Which interaction caused confusion?
