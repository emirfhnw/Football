# Evaluation Tasks

Use these tasks to evaluate whether users can answer the dashboard's main analytical questions.

## Task 1

Find the team with the longest average goal build-up.

Expected interaction: use the KPI cards and team ranking chart.

## Task 2

Identify whether quick, medium or long build-ups were most common.

Expected interaction: inspect the build-up type bar chart and optionally click a bar to filter linked views.

## Task 3

Select one Spain goal and explain the passing sequence before the goal.

Expected interaction: use the Spain case study dropdown, pitch sequence and timeline panel.

## Task 4

Compare attack duration and successful passes before goal for individual goals.

Expected interaction: use the scatterplot and hover/click on individual points.

## Task 5

Use the dashboard to assess whether finishing efficiency appears related to tournament progress.

Expected interaction: inspect the finishing efficiency vs tournament progress chart and compare team positions.

## Observation Prompts

- Did the participant understand the build-up categories?
- Did the participant notice that the pitch view is event-based, not tracking-based?
- Did linked filtering behave as expected?
- Which chart did the participant use first?
- Where did the participant hesitate?
